# Godot Kart Racing Prototype

A simple Mario Kart–style racing prototype built in Godot.
This project was created as one of my first experiments with game development and focuses on basic driving mechanics, drifting, and AI opponents.

## Overview

This prototype demonstrates the core elements of a kart racer:

* Player-controlled vehicle
* Drifting mechanics
* Basic AI opponents
* Simple track and physics

The goal of this project was to learn how to work with 3D movement, physics, and game logic in Godot.

---

## Features

* Basic kart movement (acceleration, steering, braking)
* Drift system for taking corners
* Simple AI bots that follow the track
* Lap-based racing logic (if implemented)
* Lightweight and minimal setup

---

## Built With

* Godot Engine (3D)
* GDScript

---

## Notes

This is a bare-bones prototype and not a finished game.
The focus was on learning and experimenting rather than polish.
